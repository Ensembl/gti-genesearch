To control which fields are retrieved use the fields parameter:

  • id,genome,biotype
  • id,homologues
  • id,transcripts.translations
  • id,transcripts.translations.coding*
  • Complete source

To control sort order, use the sort parameter:

  • Name, ascending
  • Name, descending
  • Biotype then name

To get a faceted breakdown of the results, use the facet parameter:

  • Facet by biotype

For basic pagination, use the limit and offset parameters:

  • First 100 entries
  • Second 100 entries

Bulk Fetch

To download entire datasets, use the /api/fetch endpoint. The Accept header can be set to application/json, text/xml or text/csv to control formatting.

  • Download all E. coli K12 MG1655 genes
  • Download gene names and homologues for genes matching the supplied UniProt accessions

Genes

For fetching complete annotation on genes, use the /api/genes endpoint. The Accept header can be set to application/json or text/xml. A POST request can be used to retrieve a list of genes.

  • Fetch BRCA2
